# Concurrent Client-Server System

## Overview
A multi-threaded client-server application built in C++.  
The system handles multiple clients concurrently using socket programming.

## Features
- Multi-threaded server
- Concurrent client handling
- Thread synchronization for safety

## Technologies
- C++
- Socket Programming
- Multi-threading

## How to Run
1. Compile the server and client
2. Run the server
3. Connect multiple clients
